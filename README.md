Prerequisites

Godot Engine (version 4.x recommended)

**Installation**

Clone the repository

bash   git clone https://github.com/anshu713-cloud/web-bundles.git
   cd "Flappy bird/web-bundles"

**Open in Godot**

Click "Import"
Navigate to the project directory

Enjoy the game!
